## Email Template Format

- Subject
- Body

Line one will be used for the **Subject**
The rest would be the **Body** and can contain simple HTML.

E.g.:
```
Welcome / Bienvenue
<p>
	Welcome to the show!
</p>
<p>
	Bienvenue au spectacle !
</p>
``` 

**IMPORTANT: The template file must be deployed as content.**
